🔒 개인정보 처리방침

시니어 긴급 의료 도우미 (Senior Medical Helper)

⚠️ 중요: 시니어 긴급 의료 도우미는 사용자의 개인정보를 최소한으로 수집하며, 수집된 정보는 기기 내부에만 저장되고 외부 서버로 전송되지 않습니다.

1. 수집하는 개인정보

1.1 위치 정보

  • 수집 목적: 사용자 주변의 가까운 병원 검색
  • 수집 시점: 앱 사용 중 (실시간)
  • 보관 방식: 기기 내부에만 일시적으로 저장
  • 보관 기간: 앱 종료 시 즉시 삭제
  • 제3자 제공: 없음 (서버로 전송하지 않음)

1.2 수집하지 않는 정보

  • 이름, 주민등록번호, 생년월일 등 개인 식별 정보
  • 전화번호, 이메일 주소 등 연락처 정보
  • 사용자의 의료 기록 또는 건강 정보
  • 결제 정보 (무료 앱)
  • 사용자 행동 패턴 또는 검색 이력

2. 개인정보의 이용 목적

수집된 위치 정보는 다음의 목적으로만 사용됩니다:

  • 사용자 현재 위치 기반 가까운 의료 기관 검색
  • 의료 기관까지의 거리 계산
  • 지도 앱 연동 시 출발지 정보 제공

3. 개인정보의 보유 및 이용 기간

위치 정보는 다음과 같이 처리됩니다:

  • 저장 위치: 사용자 기기 내부 (RAM)
  • 보유 기간: 앱 사용 중에만 메모리에 일시 보관
  • 삭제 시점: 앱 종료 또는 백그라운드 전환 시 즉시 삭제
  • 서버 전송: 없음 (모든 처리는 로컬에서만 이루어짐)

4. 개인정보의 제3자 제공

본 앱은 수집된 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우 사용자가 명시적으로 선택한 외부 앱과 최소한의 정보를 공유할 수 있습니다:

  • 지도 앱 (Google Maps, 네이버 지도, 카카오맵): 사용자가 "길찾기" 버튼을 탭할 경우, 해당 지도 앱에 출발지(현재 위치)와 목적지(병원 위치) 좌표를 전달합니다. 이는 사용자의 명시적 동의 하에 이루어지며, 지도 앱의 개인정보 처리방침이 적용됩니다.
  • 전화 앱: 사용자가 "전화걸기" 버튼을 탭할 경우, 기기의 전화 앱에 병원 전화번호를 전달합니다.

5. 개인정보의 파기 절차 및 방법

본 앱은 개인정보를 서버에 저장하지 않으므로 별도의 파기 절차가 필요하지 않습니다.

5.1 앱 삭제 시

  • 앱을 삭제하면 모든 앱 데이터가 자동으로 삭제됩니다
  • 기기 설정 > 앱 > 시니어 긴급 의료 도우미 > 데이터 삭제로도 가능합니다

5.2 위치 권한 철회

  • 기기 설정 > 앱 > 시니어 긴급 의료 도우미 > 권한 > 위치 권한 해제
  • 권한 해제 시 병원 검색 기능이 제한될 수 있습니다

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 위치 권한 거부: 앱 설치 시 또는 기기 설정에서 위치 권한을 거부할 수 있습니다
  • 부분적 사용: 위치 권한 없이도 앱의 일부 기능(전화걸기 등)을 사용할 수 있습니다
  • 앱 삭제: 언제든지 앱을 삭제하여 모든 데이터를 제거할 수 있습니다

  • 본 앱은 병원 위치 안내 서비스로, 의료 서비스를 제공하지 않습니다
