운명의 실 (Thread of Fate)

📅 최종 업데이트: 2025년 10월 21일

운명의 실(이하 "앱")은 회원가입이 필요 없는 서비스로, 이용자의 개인정보를 서버에 저장하지 않습니다. 사주팔자 및 전생 리딩을 위해 필요한 최소한의 정보만 OpenAI에 제공되며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 등 관련 법령을 준수합니다.

🔒 중요: 개인정보 비저장 정책

본 앱은 사용자의 개인정보를 서버에 저장하지 않습니다.

  • ✅ 생년월일시 정보는 기기 내부(LocalStorage)에만 저장됩니다
  • ✅ 서버에는 어떠한 개인정보도 저장되지 않습니다
  • ✅ 앱 삭제 시 모든 정보가 자동으로 삭제됩니다
  • ✅ 회원가입, 로그인, 계정 생성이 필요 없습니다

1. 수집하는 개인정보의 항목

가. 이용자가 입력하는 정보 (기기 내부 저장)

수집 항목 수집 목적 저장 위치
생년월일 사주팔자 계산 기기 내부 (LocalStorage)
출생 시간 (선택) 시주(時柱) 계산 기기 내부 (LocalStorage)
성별 (선택) 사주 해석 참고 기기 내부 (LocalStorage)

⚠️ 위 정보는 오직 이용자의 기기 내부에만 저장되며, 서버로 전송되지 않습니다.

나. 자동 수집 항목 (광고 목적)

  • 광고 ID (Google AdMob 사용 시)
  • 기기 정보 (OS 버전, 모델명 - 광고 표시용)
  • 앱 사용 통계 (광고 노출 횟수 등)

※ 광고 ID는 Google AdMob이 수집하며, 맞춤형 광고 제공에만 사용됩니다.

2. 개인정보의 수집 및 이용 목적

가. 서비스 제공 (OpenAI 전송)

🔄 OpenAI에 전송되는 정보:

  • 생년월일: 사주팔자 계산 및 해석
  • 출생 시간: 시주(時柱) 계산 (입력한 경우)
  • 성별: 운세 해석 참고 (선택 입력한 경우)

전송 목적:

  • 사주팔자 분석 및 운세 해석 생성
  • 전생 리딩 스토리 생성
  • 오늘의 운세 생성
  • 운명 바꾸기 (긍정적 재해석) 생성

⚠️ 중요: OpenAI는 API 요청 처리 후 30일 내에 데이터를 삭제합니다. (OpenAI 개인정보처리방침 참조)

나. 광고 제공 (Google AdMob)

  • Google AdMob을 통한 광고 표시
  • 광고 ID를 통한 맞춤형 광고 제공
  • 광고 효과 측정

※ 광고 관련 정보는 Google이 수집하며, Google의 개인정보처리방침을 따릅니다.

3. 개인정보의 보유 및 이용 기간

✅ 서버 비저장 정책:

  • 본 앱은 서버에 개인정보를 저장하지 않습니다
  • 모든 정보는 이용자의 기기 내부(LocalStorage)에만 저장됩니다
  • 앱을 삭제하면 모든 정보가 자동으로 삭제됩니다

가. 기기 내부 저장 정보

  • 저장 위치: 기기 내부 (LocalStorage)
  • 보유 기간: 앱 삭제 시까지
  • 삭제 방법:
    • 앱 삭제 (즉시 삭제)
    • 앱 내 "정보 변경" 메뉴에서 재입력 (덮어쓰기)
    • 기기 설정 → 앱 데이터 삭제

나. OpenAI 전송 정보

  • 보유 기간: 요청 처리 후 30일 이내 삭제 (OpenAI 정책)
  • 삭제 주체: OpenAI LLC
  • 참고: OpenAI 개인정보처리방침

다. 광고 ID (AdMob)

  • 수집 주체: Google LLC
  • 보유 기간: Google 정책에 따름
  • 재설정 가능: 기기 설정에서 광고 ID 재설정 가능

4. 개인정보의 제3자 제공 ⭐ 중요

⚠️ 본 앱은 서비스 제공을 위해 다음과 같이 개인정보를 제3자에게 제공합니다:

가. OpenAI LLC (필수 제공)

항목 내용
제공받는 자 OpenAI LLC (미국)
제공 목적 사주팔자 분석, 전생 리딩, 운세 해석 생성
제공 항목 • 생년월일
• 출생 시간 (입력한 경우)
• 성별 (선택 입력한 경우)
• 사주팔자 계산 결과 (천간, 지지)
보유 기간 요청 처리 후 30일 이내 삭제
제공 방식 HTTPS 암호화 통신 (TLS 1.3)
거부 권리 제공을 거부할 수 있으나, 서비스 이용 불가

📌 OpenAI 개인정보처리방침: https://openai.com/policies/privacy-policy

나. Google LLC (AdMob - 광고)

항목 내용
제공받는 자 Google LLC (미국)
제공 목적 광고 표시 및 맞춤형 광고 제공
제공 항목 • 광고 ID (GAID/IDFA)
• 기기 정보 (OS, 모델명)
• 앱 사용 통계
보유 기간 Google 정책에 따름
거부 방법 기기 설정에서 맞춤 광고 끄기 가능

📌 Google 개인정보처리방침: https://policies.google.com/privacy

✅ 안전한 전송:

  • 모든 정보는 HTTPS(TLS 1.3) 암호화 통신으로 전송됩니다
  • 이름, 이메일, 전화번호 등 직접 식별 가능한 정보는 수집하지 않습니다
  • 생년월일만으로는 특정 개인을 식별할 수 없습니다

5. 개인정보 처리의 위탁

수탁업체 위탁 업무 내용
OpenAI LLC AI 모델 기반 운세 콘텐츠 생성
Google LLC 광고 송출 및 분석
클라우드 서비스 제공업체 데이터 저장 및 서버 운영

6. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

가. 개인정보 열람

  • 앱 내 "정보 변경" 메뉴에서 확인 가능
  • 기기에 저장된 정보만 표시 (서버 저장 없음)

나. 개인정보 수정

  • 앱 내 "정보 변경" 메뉴에서 재입력 가능
  • 즉시 반영됩니다 (서버 동기화 불필요)

다. 개인정보 삭제

  • 방법 1: 앱 삭제 (모든 정보 자동 삭제)
  • 방법 2: 기기 설정 → 앱 데이터 삭제
  • 효과: 기기 내 모든 정보 즉시 삭제

라. OpenAI 전송 정보 삭제

  • OpenAI는 요청 처리 후 30일 내 자동 삭제
  • 즉시 삭제 요청: privacy@openai.com

마. 광고 맞춤설정 거부

  • Android: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제
  • 효과: 맞춤형 광고 대신 일반 광고 표시

7. 개인정보의 파기

가. 기기 내부 저장 정보

  • 파기 방법: 앱 삭제 시 자동 파기
  • 파기 시점: 이용자가 앱 삭제 시 즉시
  • 파기 주체: 이용자 (직접 삭제)
  • 복구 불가: 삭제 후 복구 불가능

나. OpenAI 전송 정보

  • 파기 방법: OpenAI 시스템에서 자동 삭제
  • 파기 시점: 요청 처리 후 30일 이내
  • 파기 주체: OpenAI LLC

다. 광고 ID

  • 재설정: 기기 설정에서 광고 ID 재설정 가능
  • 효과: 새로운 ID 발급, 기존 데이터 연결 끊김

✅ 서버 비저장 정책의 장점:

  • 별도의 파기 절차 불필요
  • 앱 삭제만으로 모든 정보 삭제
  • 서버 해킹 위험 없음
  • 개인정보 유출 위험 최소화

8. 개인정보 보호를 위한 대책

가. 기술적 대책

  • 암호화: 생년월일 등 민감 정보 AES-256 암호화 저장
  • HTTPS: 모든 통신 구간 TLS 1.3 암호화
  • 방화벽: 외부 침입 차단 시스템 운영
  • 백업: 정기적 데이터 백업 및 복구 체계

나. 관리적 대책

  • 개인정보 취급자 최소화 및 교육
  • 개인정보 접근 권한 관리
  • 정기적인 보안 점검 및 모니터링
  • 내부 관리 계획 수립 및 시행

9. 쿠키(Cookie) 및 광고 ID

가. 쿠키의 사용 목적

  • 이용자의 접속 빈도 및 서비스 이용 패턴 파악
  • 맞춤형 서비스 제공
  • 서비스 개선 및 추천 서비스 제공

나. 광고 ID 사용

  • Google 광고 ID 사용 (Android)
  • 맞춤형 광고 제공 및 광고 효과 측정
  • 설정에서 광고 ID 재설정 또는 맞춤 광고 거부 가능

다. 광고 ID 거부 방법

  • Android: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제
